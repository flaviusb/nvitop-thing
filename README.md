# nvitop-thing
A small script for writing out gpu usage data using nvitop

## To install this

```bash
git clone https://github.com/flaviusb/nvitop-thing.git
cd nvitop-thing
virtualenv foo
. foo/bin/activate
pip3 install --upgrade nvitop
```

Then, later, to run you can run:

```bash
cd nvitop-thing
. foo/bin/activate
python3 logging-script-thing.py
```
